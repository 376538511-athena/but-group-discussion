## 四、一键部署流程

```bash
# 1. 登录云服务器，安装 Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 2. 克隆项目
git clone <your-repo-url> /opt/seminar
cd /opt/seminar

# 3. 配置生产环境变量
cp .env.example .env.production
vim .env.production  # 修改数据库密码、JWT 密钥等

# 4. 一键启动
docker compose -f docker-compose.prod.yml --env-file .env.production up -d

# 5. 创建管理员账号
docker compose -f docker-compose.prod.yml exec server node dist/seed.js

# 6. 配置 SSL（首次）
docker compose -f docker-compose.prod.yml exec nginx certbot --nginx -d your-domain.com

# 7. 验证
curl https://your-domain.com/api/v1/health
```

## 五、域名与 SSL 方案

**方案 A：自有域名（推荐）**
- 购买域名（阿里云/腾讯云，.com 约 55元/年）
- DNS 解析 A 记录指向服务器 IP
- Let's Encrypt 免费 SSL（自动续期）

**方案 B：无域名**
- 直接用 IP + 端口访问（http://1.2.3.4:80）
- 不支持 HTTPS（浏览器会提示不安全）
- 仅适合内网或临时测试
